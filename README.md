# Netflix Data Analysis Dashboard (Power BI)

## Project Overview
This project analyzes the Netflix dataset using Power BI to understand content trends, genre distribution, ratings, and release patterns.

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization

## Dataset
Netflix Movies and TV Shows dataset.

Columns include:
- Title
- Type (Movie / TV Show)
- Country
- Release Year
- Rating
- Duration
- Genre

## Dashboard Insights
The dashboard provides insights such as:

- Distribution of Movies vs TV Shows
- Most common genres on Netflix
- Content growth over the years
- Top countries producing Netflix content
